# Full stack GraphQL with Apollo

1. Start the GraphQL backend:

```bash
cd backend
yarn install
yarn start
```

2. Start the React/TypeScript frontend:
```bash
cd frontend
yarn install
yarn start
```

3. The app is running:

* Frontend: `http://localhost:9080`
* GraphiQL: `http://localhost:9000/graphiql`
